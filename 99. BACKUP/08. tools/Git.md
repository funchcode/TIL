# Git

## 자주쓰는 명령어 정리

### add

* 모든 변경 파일
  * ```git add .```
  * ```git add -A```
* 선택한 파일 여러 개
  * ```git add [파일명] [파일명] ... [파일명]```
* 특정 폴더에 위치하는 파일
  * ```git add [폴더명]```
* diff 확인 후 스테이징
  * ```git add -p```
> 변경된 코드들의 파일을 하나씩 보여주는데, 특정 키워드로 파일을 선택할  수 있다.  
> [*키워드] y: 스테이지에 올리기, n: 다음 파일, q: 나가기
