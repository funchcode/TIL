# 예외

## C2065 선언되지 않은 식별자

상황  

- string 타입을 사용하기 위해서 `#include <string.h>`을 선언하고 string을 타입으로 선언.
- C2065 선언되지 않은 식별자 오류가 발생

해결

- `using namespace std;`를 선언하여 현상 해결

using namespace std

- 이름의 충돌을 막아주고 프로그램 가독성을 향상시키기 위해 사용한다.
- 충돌
  - C++에는 표준 라이브러리가 존재한다. (자료구조, 알고리즘, 입출력 등...)
    - STL (Standard Template Library)
  - 다른 라이브러리에서 표준 라이브러리의 함수 이름, 클래스 이름 등이 동일한 경우 충돌이 발생한다.
