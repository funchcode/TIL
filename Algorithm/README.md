# Algorithm

프로그래밍 대회에서 배우는 알고리즘 문제 해결 전략 (구종만 지은)

파인만 알고리즘

1. 칠판에 문제를 적는다.
2. 골똘히 생각한다.
3. 칠판에 답안을 적는다.

재정의

1. 문제를 읽고 이해한다.
2. 문제를 익숙한 용어로 재정의한다.
3. 어떻게 해결할지 계획을 세운다.
4. 계획을 검증한다.
5. 프로그램으로 구현한다.
6. 어떻게 풀었는지 돌아보고, 개선할 방법이 있는지 찾아본다.

회고

1. 해법
2. 문제 접근 방식
3. 문제 해결에 결정적인 깨달음
4. 오답 원인
5. 다른 사람의 문제 접근 방식 분석
