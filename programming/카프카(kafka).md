Apache Kafka
3가지 주요 기능
1) 다른 시스템 데이터 지속적으로 가져오기/내보내기, 이벤트 스트림 Publish, Subscribe
2) 설정한 기간 동안 이벤트 스트림을 지속적이고 안정적으로 저장
3) 이벤트 스트림 발생 시 소급하여 처리
